Function jsdocTransformer

  • jsdocTransformer(handler): ts.TransformerFactory<ts.Bundle | ts.SourceFile>

  • Provides a convenient "meta-transformer" that invokes a handler function for each Node w/ the parsed leading comment data for the Node. Only leading block comments are parsed. The parsed array is in the data format provided by the comment-parser package. For convenience there are lastComment and lastParsed fields that return the last comment block respectively before the node. Typically, the last comment is the active JSDoc block for a Node.

    Note: In the handler return null to remove the Node.
